util module

This module contains utility functions like list handling, memory management, etc. For compatibility with the glib, all functions starts with a "g_" prefix.

This library is an extended subset of the glib library.
The glib is the base library of GTK+ (www.gtk.org).
It is release under the term of the GNU LGPL (www.gnu.org/copyleft/lesser.html).
Many functions have been removed or rewritten.
A few functions have been added.
